# بوابة القبول في كليات التدريب المهني والتقني

نظام إلكتروني متكامل للقبول والتسجيل في الكلية التقنية، يتيح للمتقدمين (خريجين، تجسير، منقولين) تقديم طلباتهم بسهولة مع ميزة التحقق من البيانات والإرسال المباشر للبريد الإلكتروني.

## مميزات النظام:
* **واجهة مستخدم حديثة:** متوافقة مع جميع الأجهزة والجوالات.
* **إرسال مباشر:** ربط الطلبات ببريد إدارة القبول فوراً.
* **خيارات متعددة:** دعم مسارات التقديم المختلفة (ثانوي/تجسير).
